# AlgorithmsPHP
Some Algorithms written in PHP for language studies

## Requirements

- PHP 5.5+
- Composer

## Installation

```shell
$ composer install && composer dump-autoload 
```

## Algorithms

| Name 						          | Solutions 								            | Description 										                  |
|---------------------------|---------------------------------------|---------------------------------------------------|
| ReverseString			        | 1 								                    | Reverse a given string            								|       

Dependencies
------------

None


Run tests
----------------

Inside a root directory run that command:

```shell
$ vendor/bin/phpunit
```
